# 🧮 JavaScript Calculator (Tailwind CSS)

A simple and modern calculator web app built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**.  
It performs basic arithmetic operations and provides a sleek, responsive user interface.

---

## 🚀 Features

- ✨ Built entirely with **Tailwind CSS** (no custom CSS needed)
- ⚡ Performs basic operations: `+`, `-`, `×`, `÷`, `%`
- 🔁 Toggle sign `±` support
- 🧹 Clear (`C`) and decimal (`.`) functionality
- 📱 Fully responsive and mobile-friendly UI
- 🧠 Handles invalid inputs gracefully with an `Error` message

---

## 🧰 Tech Stack

| Technology | Purpose |
|-------------|----------|
| **HTML5** | Structure of the app |
| **Tailwind CSS** | Styling & layout |
| **JavaScript (ES6)** | Logic and interactivity |
